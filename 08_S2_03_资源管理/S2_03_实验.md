# S2_03_实验 — 资源管理

---

## 实验 1：/proc/iomem 读取

```bash
cat /proc/iomem | grep "PCI"
```

看 PCI MMIO 地址分配。

---

## 实验 2：BAR 分配查看

```bash
cat /sys/bus/pci/devices/0000:01:00.0/resource
### 对比：
cat /sys/bus/pci/devices/0000:01:00.0/ioport_resource
```
