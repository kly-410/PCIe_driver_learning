# S2_04_实验 — MSI/MSI-X 中断路由

---

## 实验 1：查看设备 MSI 能力

```bash
### 查看 01:00.0 的 MSI Cap
sudo lspci -nn -s 01:00.0 | grep -i msi
cat /sys/bus/pci/devices/0000:01:00.0/msi_interrupts_count
```

---

## 实验 2：MSI-X 向量数查看

```bash
cat /sys/bus/pci/devices/0000:01:00.0/msi_irqs
### 显示已分配的 MSI 中断向量数
```
